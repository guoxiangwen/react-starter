# react-starter
react+webpack+es6+less+babel
# how to start
1. `npm install --安装`(注意:chromedriver如果没有翻墙的话，是安装不成功的,需要单独安装 `npm install chromedriver --chromedriver_cdnurl=https://npm.taobao.org/mirrors/chromedriver`)
2. `npm test --测试`
3. `npm run build --发布`
